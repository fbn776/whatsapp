Guide for adding custom list


You can add custom list to the existing lits. When you choose the random generation option, you can chose from these options: A-Z, a-Z, 0-9, and some special characters. Using this you add custom ones.
To add these, you need to enter the list element one by one and seperate them using commas (,).

Example:
You need a set of names to display that names in random order in a chat, for that you can do this,

Name1,Name2,Name3,Name4...

These value are stored, so whenever you come back, those list will be saved!
Note: Some characters are larger than they appear. So even when you are selecting a long lengthed item, the message won't be that long.
